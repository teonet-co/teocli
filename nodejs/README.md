# Teonet client nodejs module [![NPM version][npm-image]][npm-url]

## Install with bower:  
```
bower install teonet-client
```

## Install with npm:  
```
npm install teonet-client
```

## Notes

Use before build this package:

    sudo apt-get install build-essential
